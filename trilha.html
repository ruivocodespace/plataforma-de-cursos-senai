Siga a trilha do começo ao fim

Construa o painel admin primeiro — assim você terá cursos, módulos e aulas cadastrados para testar as telas do aluno. Marque os itens conforme for fazendo. Ao terminar tudo, você terá o sistema EAD completo e funcional.

Você já sabe fazer isso! O padrão é o mesmo da aula 5: mysqli_query, $_SESSION, $_POST, $_GET, logado.php, conexao.php. Só muda o contexto. No admin, o CRUD usa arquivos separados: *_form.php para cadastrar/editar e *_delete.php para excluir.

1
Etapa 1 — Login login.php — necessário para acessar o admin

O login precisa existir antes do admin — você vai precisar dele para entrar no painel e cadastrar os dados. Copie da aula 5 e adapte.

2
Etapa 2 — Admin: Dashboard admin/index.php

Dashboard simples que mostra os totais. Toda página dentro de admin/ começa com a verificação de sessão.

3
Etapa 3 — Admin: CRUD de Cursos cursos.php · curso_form.php · curso_delete.php

O CRUD usa três arquivos separados. A lógica do formulário é a mesma do cadastro_produto.php da aula 5.

admin/cursos.php — Listagem

admin/curso_form.php — Cadastrar / Editar

admin/curso_delete.php — Excluir

4
Etapa 4 — Admin: CRUD de Módulos e Aulas modulo_form · aula_form · *_delete

Mesma estrutura da Etapa 3. Copie os 3 arquivos de cursos e ajuste a tabela e os campos extras.

Módulos — 3 arquivos

Aulas — 3 arquivos

Checkpoint — Insira Dados Reais Antes de Continuar!

A área do aluno só terá conteúdo se o admin já tiver cadastrado cursos, módulos e aulas.

  1. Logar com admin@ead.com / senai123
  2. Cadastrar pelo menos 2 cursos com título e descrição
  3. Para cada curso, cadastrar pelo menos 2 módulos
  4. Para cada módulo, cadastrar pelo menos 2 aulas com URL de vídeo
  5. Confirmar no phpMyAdmin que as tabelas têm dados

Só avance para a Etapa 5 quando o painel admin estiver funcionando e com dados inseridos.

5
Etapa 5 — Cadastro do Aluno cadastro.php

Formulário público. Igual ao cadastro de usuário da aula 5 — campos nome, email e senha.

6
Etapa 6 — Proteger as Páginas do Aluno logado.php — igual à aula 5

Todo arquivo interno do aluno deve incluir o logado.php no topo — exatamente como na aula 5.

Vale nota: toda página interna precisa ter essa proteção. Páginas sem logado.php perdem pontos.
7
Etapa 7 — Catálogo de Cursos e Inscrição cursos.php

Agora que o admin já cadastrou cursos, o aluno pode vê-los e se inscrever. Para cada curso, o sistema verifica se já está inscrito para mostrar o botão certo.

8
Etapa 8 — Meus Cursos meus_cursos.php

Página pessoal. Lista só os cursos em que o aluno está inscrito, com botão "Acessar".

9
Etapa 9 — Módulos e Player de Aula curso.php · aula.php

Dois arquivos simples: curso.php lista os módulos e aulas do curso; aula.php exibe o player de vídeo.

curso.php — Módulos e Aulas do Curso

aula.php — Player

10
Etapa 10 — Testar Tudo e Entregar Finalização

Percorra o sistema nos dois perfis e confirme que tudo funciona antes de compactar.
