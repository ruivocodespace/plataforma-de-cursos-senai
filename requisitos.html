O que é o Sistema

Você vai criar uma plataforma de Ensino a Distância (EAD) com dois perfis de usuário:

Aluno

Se cadastra, faz login, se inscreve em cursos e navega pelos módulos e aulas.

📚

Cursos

Cada curso tem módulos, e cada módulo tem aulas cadastradas pelo admin.

Admin

Cadastra, edita e exclui cursos, módulos e aulas. Mesmo CRUD da aula 5.

Fluxo do aluno: Tela inicial → Cadastro → Login → Ver cursos → Se inscrever → Acessar módulos → Ver aulas

2
O que o Sistema Precisa Ter
RF01 Login e Logout Igual à aula 5
  • Tela de login com e-mail e senha (copie o login.php da aula 5).
  • Buscar usuário com mysqli_query e verificar com password_verify().
  • Salvar ID, nome e tipo do usuário em $_SESSION.
  • Logout destrói a sessão e redireciona para a tela inicial.
RF02 Cadastro de Aluno
  • Formulário público (sem login) com: nome, e-mail e senha.
  • Verificar se o e-mail já existe: SELECT + mysqli_num_rows.
  • Salvar senha com password_hash() — igual ao que já foi feito.
  • Após cadastro, redirecionar para o login.
RF03 Catálogo de Cursos e Inscrição
  • Listar todos os cursos: SELECT * FROM cursos.
  • Verificar se o aluno já está inscrito (consulta na tabela inscricoes).
  • Se não inscrito: botão "Inscrever-se" que insere na tabela inscricoes.
  • Se inscrito: botão "Acessar" que leva para os módulos do curso.
RF04 Meus Cursos — Ver Módulos e Aulas
  • Página que lista só os cursos em que o aluno está inscrito.
  • Em curso.php?id=N listar os módulos: SELECT * FROM modulos WHERE curso_id = $id.
  • Em aula.php?id=N mostrar o player do vídeo e os dados da aula.
  • Cada aula exibe título, duração e o iframe/link do vídeo.
RF05 Painel Admin — CRUD de Cursos, Módulos e Aulas Igual ao CRUD da aula 5
  • Acesso restrito: $_SESSION["usuario_tipo"] == 'admin'.
  • Cursos: admin/cursos.php lista · admin/curso_form.php cadastra/edita · admin/curso_delete.php exclui.
  • Módulos: admin/modulos.php lista · admin/modulo_form.php cadastra/edita · admin/modulo_delete.php exclui.
  • Aulas: admin/aulas.php lista · admin/aula_form.php cadastra/edita · admin/aula_delete.php exclui.
  • Os forms recebem ?id=N para editar ou ficam vazios para cadastrar — mesma lógica da aula 5.

Bônus (não obrigatório): Upload de imagem de capa para cursos · Marcar aula como concluída · Barra de progresso

Banco de Dados

Importe projeto/sql/banco.sql no phpMyAdmin. As tabelas são simples e diretas:

Tabela Campos principais Para que serve
usuarios id, nome, email, senha, tipo Alunos e admins. Campo tipo: 'aluno' ou 'admin'
cursos id, titulo, descricao, capa Catálogo de cursos disponíveis
modulos id, curso_id, titulo, ordem Módulos (capítulos) de cada curso
aulas id, modulo_id, titulo, video_url, duracao Aulas de cada módulo
inscricoes id, usuario_id, curso_id Quais alunos estão inscritos em quais cursos

Admin padrão após importar o banco: E-mail: admin@ead.com  ·  Senha: admin123

Causa desconto de nota

  • Página interna acessível sem login
