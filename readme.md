# Plataforma de Cursos EAD SENAI - HTML, PHP + MySQL 🚀

> 🚧 Projeto em desenvolvimento – novas funcionalidades estão sendo implementadas continuamente.

---

## 🛠 Tecnologias Utilizadas

- PHP
- MySQL
- HTML
- CSS (Tailwind)
- Controle de sessão (login/logout)
- Integração de video do youtube

---

## 🚀 Funcionalidades

- Cadastro, login e logout
- Controle de sessão
- Cadastro de cursos, módulos e aulas.
- Cadastro de curso com upload de imagem
- Cadastro de aulas com link de reprodução (embed)
- Integração com banco de dados local
- Interface estruturada com CSS Tailwind

---

## 🧠 Conceitos Praticados

- CRUD completo
- Integração PHP + MySQL
- Manipulação de arquivos (upload)
- Autenticação de usuários
- Organização modular de código
- Estruturação básica de sistema web
- Controle de versionamente no com Git/GitHub

---

## ▶️ Como Executar

1. Instale um servidor local (XAMPP, WAMP ou similar)
2. Importe o arquivo `banco.sql` no MySQL
3. Configure as credenciais no arquivo `conexao.php`
4. Execute o projeto no navegador
